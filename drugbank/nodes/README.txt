Generate Nodes from Drugbank Database
-------------------------------------

1. Takes as input a single .xml file from the drugbank database to 
   generate node mapping.
2. Steps: 
   I. Download .xml in the current directory.
   II. Run make-nodes.sh
3. If BeautifulSoup is not installed or you cannont install:
   I. Follow steps mentioned on http://docs.python-guide.org/en/latest/dev/virtualenvs/ 
   II. Next launch the env as : source /env/bin/activate
   III. Run make-nodes.sh
